# ActixLab

Shared Rust backend learning repo for collaborative Actix and systems projects.

## Structure

- `rust-backend-lab/` contains the 50 guided backend tracks.
- Each numbered folder represents one track/project area.
- Existing runnable Rust crates can live directly inside a track folder.
- Shared utilities can live in `rust-backend-lab/common/`.

## Collaboration workflow

- Do not work directly on `main`.
- Create a branch per task, for example: `frank/02-crud-api` or `mike/03-auth-jwt`.
- Pull latest `main` before starting new work.
- If two people are working on the same track, create separate subfolders inside that track or agree on file ownership before editing.

See `CONTRIBUTING.md` for the team workflow.
